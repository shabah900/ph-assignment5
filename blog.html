Important FAQ

What is the Document Object Model (DOM)?

Ans : The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, allowing programming languages like JavaScript to access, manipulate, and modify the content, structure, and style of the page dynamically.

How do you select an element from the DOM?

Ans : We can select elements from the DOM using JavaScript methods such as:

1.document.getElementById('id') – Selects an element by its ID.

2.document.querySelector('selector') – Selects the first element that matches a CSS selector.

3.document.getElementsByClassName('class') – Selects elements by their class name.

What is event delegation in the context of the DOM, and why is it useful?

Ans : Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events triggered by its child elements. This is useful because it allows for better performance and cleaner code, especially when dynamically adding or removing elements.

How do you manipulate an element's attributes and styles using the DOM?

Ans : For attributes: element.setAttribute('attribute', 'value') or directly through properties like element.src for images or element.href for links. For styles: You can use element.style.property = 'value', for example, element.style.color = 'red' to change the text color.
